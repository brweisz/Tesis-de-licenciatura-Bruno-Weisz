# Tesis-de-licenciatura-Bruno-Weisz

En este repositorio se encuentran los principales componentes de la tesis de licenciatura de Bruno Weisz. Estos son:

* ```implementacion```: contiene un readme con instrucciones para instalar y usar la herramienta.
* ```experimentacion```: contiene los experimentos y resultados obtenidos en cuanto a mediciones de performance.
* ```reporte```: contiene el PDF con el reporte de la tesis, incluyendo todos los archivos .tex que la componen.
